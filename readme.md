# Git Config

## Installation

- Set user and email according to the directory
- Ignore files globally

## Installation
```bash
git clone git@gitee.com:baijunyao/gitconfig.git ~/gitconfig
```

## Usage
Add the following code to the ~/.gitconfig
```
[include]
    path = ~/gitconfig/.gitconfig
```
